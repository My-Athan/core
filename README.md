# My-Athan Backend

Monorepo containing the backend API, admin dashboard, and website for the My-Athan smart device platform.

## Repository Structure

```
backend/
├── apps/
│   ├── api/              # Backend API service
│   ├── admin/            # Admin dashboard (web)
│   └── web/              # Marketing site + user portal
├── packages/
│   └── shared/           # Shared types, utils, constants
├── infra/                # Docker, CI/CD, deployment configs
├── docs/                 # Architecture docs, API specs
└── .github/              # GitHub Actions workflows
```

## Related Repositories

| Repository | Description |
|------------|-------------|
| `firmware` | Smart device firmware and embedded code |
| `app`      | Cross-platform mobile app (iOS & Android) |
